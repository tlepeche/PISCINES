<- Vertex -----------------------------------------------------------------------------
the Vertex class handles x, y, z, w vertex and RGB colors.

The instance must be construct with x, y and z vertex :
new Vertex ( array ( 'x' => 1.0, 'y' => 1.0, 'z' => 1.0) );

w and color are optionals (default : w = 1.0 and color is white).

For more information about color class, refer to the class doc.
